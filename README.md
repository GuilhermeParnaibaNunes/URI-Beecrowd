# 📚 URI / Beecrowd Solutions

A collection of solutions to problems from the [Beecrowd](https://www.beecrowd.com.br/) (formerly URI Online Judge), primarily implemented in **C and C++**.

---

## 🚀 Purpose

This repository was created to:

* Practice problem-solving and algorithmic thinking
* Strengthen knowledge of **data structures and logic**
* Improve proficiency in **C/C++**
* Track progress through Beecrowd problem sets

---

## 📂 Repository Structure

Problems are organized by category and/or difficulty:

```
/1. BEGINNER
/2. AD-HOC
...
```

Each folder contains solutions identified by their **problem number and name**.

---

## 🛠️ Technologies Used

* C
* C++
* Standard libraries (stdio, stdlib, iostream, etc.)

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/GuilhermeParnaibaNunes/URI-Beecrowd.git
```

2. Navigate to a problem folder:

```bash
cd URI-Beecrowd/1.\ BEGINNER
```

3. Compile and run:

```bash
gcc problem.c -o program
./program
```

> For C++ files:

```bash
g++ problem.cpp -o program
./program
```
